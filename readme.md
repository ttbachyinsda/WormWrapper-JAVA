#### 运行方法

在pom.xml所在的文件夹下：

mvn clean compile

mvn exec:java -Dexec.mainClass="Main"

参数设置：

Main.java中，有一个MongoDB连接字符串的设置，可以依情况进行修改

ThreadPool.java中，有一个设置线程数量的参数

在Main.java中，可以打开与之前python对应的方法。注意某些方法会传输大量的数据。
